# bluetoothctl
#bluetoothctl #fedora #redhat #rhel
```
$ bluetoothctl
# list     --> lists available bluetooth controllers
# select   --> select a bluetooth controller if multiple available
# show     --> shows selected bluetooth controller info
# power on/off   --> power on/off selected bluetooth controller

# devices Trusted
# devices Paired

# scan on/off

# pair <mac>
# cancel-pairing <mac>

# connect <mac>
```
